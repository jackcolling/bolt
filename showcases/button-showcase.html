Implementation Notes

Key CSS properties for Bolt buttons:

  • • border-radius: 5px for standard corners
  • • padding: 16px 40px for standard size
  • • font-weight: 700 for primary, 600 for secondary
  • • transition: all 0.2s ease for smooth interactions
  • • transform: translateY(-2px) on hover for lift effect
  • • box-shadow optional on hover for depth
  • • Yellow (#E1FF00) inverts to black (#11190C) on hover
  • • Always use Inter font family with fallbacks
